
# CzechActiveAuras

Czech translation of Active Auras version 0.12

## Popis
Tento modul poskytuje českou lokalizaci pro Active Auras verze 0.12. Obsahuje překlady základních textů aplikace.

## Instalace
Pro instalaci modulu obvyklým upůsobem můžeš použít následující odkaz ve Foundry VTT:

```
https://raw.githubusercontent.com/Lethrendis/CzechTranslationActiveAuras/main/module.json
```
